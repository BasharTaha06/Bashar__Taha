About

I'm Bashar Taha, a software engineer based in Cairo, Egypt, with a B.Sc. in Computer Science from Assiut University. I started out building mobile apps in Flutter, and that's still where I feel most at home — but I didn't want to stop at the UI, so I picked up .NET Web API for the backend, until I could build a full product on my own, front to back.

I hold an ICPC ECPC Honorable Mention, so I don't just write code that works — I think about how it scales. That competitive-programming background shows up in every backend I design.

The project I'm proudest of isn't the most technically complex one — it's Autism Helper, an Arabic-first app I built to help autistic children practice facial expressions and speech. It's a reminder to myself that the point of writing code is to actually help someone.

I'm now looking for full-time roles and freelance work — anything from a quick MVP to a product I can own end to end.
